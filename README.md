# Node.js Backend API

> RESTful API для управления товарами и пользователями с аутентификацией JWT

## Функциональность

- JWT токены, регистрация, логин
- CRUD операции
- Загрузка файлов - поддержка загрузки изображений
- PostgreSQL с Sequelize ORM
- хеширование паролей, CORS, валидация

## Технологии

- Node.js, Express.js, JavaScript, PostgreSQL,
- Sequelize ORM, JWT, bcrypt, express-fileupload,  
- CORS, dotenv для переменных окружения
